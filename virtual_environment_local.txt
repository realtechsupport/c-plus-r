Create virtual environement:


1)  get the file local_python+git.sh file (in Deploy and on Github)
2)  make it executable (chmod +x local_python+git.sh)
3)  run it: sh local_python+git.sh
4)  move around dirs as you like..
5)  create a virtual environment: python3 -m venv env
5a) cd to the where you installed the environment...
6)  active the virtual environment: source ./env/bin/activate
7)  install requirements: pip3 install -r requirements.txt
8)  test project
